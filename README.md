У нас есть набор продуктов (products).

Нужно из них сформировать таблицу (использовать тег table, thead, tbody и т д).
Внимательно изучи структуру массива products  и подумай как будешь формировать таблицу.
Пока что начнём с того, что просто сформируем таблицу с помощью js. Стили у таблицы произвольные,
но более менее чтобы адекватные были.

Формирование таблицы пиши в функции createProductsTable, которая будет использовать полученные продукты через параметр
и заголовки для таблицы, полученные из функции getTableHeaders.

Изучи уже написанные код и делай по такой структуре.

Когда справишься - будем усложнять.

Примечания ко всему проекту:
* разбивай функциональность на отдельные функции и модули
(как пример увидишь я вынес логику получения продуктов отдельно, логику формирования таблицы отдельно и
все это это используется в главном index.js файле). Используй импорты и экспорты.
* тут подключен eslint. Каждый пропущенный пробел или точка с запятой будет считаться ошибкой.
Бывает, что на время разработки удобно его отключить. Нужно просто в начале файла написать  /* eslint-disable */
Но когда будешь заливать код в репозиторий, то не забывай удалять эти комментарии и проверяй, чтобы  
eslint не ругался перед тем как комитить.
* Заливай свою работу мелкими частями, деля ее на мелкие таски, например "Add table styles"

Запустить проект:
* yarn (один раз чтоб установить нужные зависимости)
* yarn start (каждый раз для запуска)
